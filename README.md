# Date Calculator

A standalone date calculator with no ads.

Hosted at https://date-calculator.surge.sh

## Features

- **Count Date** - Calendar days, working days, and non-working days between two dates
- **Add/Subtract Date** - Add or subtract days/weeks/months/years; toggle workday mode to skip weekends + holidays
- **Week #** - Find the week number for any date
- **Holidays** - Optional list of full-day (`YYYY-MM-DD`) or half-day (`YYYY-MM-DD h`) holidays for workday calculations
